# Chuong 3 

## Bai 1 

```assembly
ORG  0000H
MOV  A, #07FH   ; A = 07FH 
SETB C          ; C = 1 
RLC  A          ; A = 3FH, C = 0
MOV  R1, #30H   ; R1 = 30H
MOV  30H, #89H  ; 30H = 89H
MOV  @R1, A     ; 30H = 3FH
ADD  A, R1      ; A = 81H 
ORL  A, 30H     ; A = D1H
SWAP A          ; A = D1H
END
```

1. A   = D1H
2. R1  = 30H  
3. 30H = 3FH
